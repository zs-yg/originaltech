# OriginalTech 原版科技

一个简易、实用、易合成的 Slimefun4 附属插件，致力于使用 Minecraft 原版材料。

## 理念

- **简易**：简洁的物品系统，没有复杂的科技树
- **实用**：每个物品都有其用途，帮助玩家更方便地进行原版游戏
- **易合成**：配方简单，材料容易获取
- **原版材料**：优先使用 Minecraft 原版材料，减少对稀有资源的依赖

## 前置要求

- Minecraft 1.21+
- Paper 或 Spigot 服务器
- Slimefun4 RC-36 或更高版本

## 安装

1. 下载 `OriginalTech-1.0.6.jar`
2. 将 jar 文件放入服务器的 `plugins` 文件夹
3. 重启服务器
4. 确保 Slimefun4 已安装并正常运行

## 开发

### 环境要求

- JDK 17
- Maven 3.6+

### 编译

```bash
mvn clean package
```

编译后的 jar 文件位于 `target/OriginalTech-1.0.0.jar`
